## Requirements

Java Runtime


## How to Build

``` bash
chmod +x gradlew

./gradlew clean build

chmod +x ./build/libs/run.sh
```

## How to Use


``` bash
./build/libs/run.sh --help

usage: ./run.sh
 -c,--config <arg>   Configuration file
 -h,--help           Prints this help
 -i,--input <arg>    Input CSV file
```
